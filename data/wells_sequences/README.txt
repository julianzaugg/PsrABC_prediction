README
Files with MopBfamily.fasta contain the genomic sequences we gathered through DELTA-BLAST searches.  Sequence headers include the annotated enzyme name, the organism the genome came from, as well as the IMG locus tag.
Files iwth MopBfamilywcodes.fasta contain the genomic sequences we gathered through DELTA-BLAST searches.  Sequence headers correspond to those found in the phylogenetic trees from iTOL.
Files with MopBfamilyfinal.fasta contain the metagenomic sequences we gathered through BLASTX searches.  Sequence headers contain the GTDB-TK identifier as well as the DNA coordinates within the MAG.
DMSORcompwoutMop.fasta contains all the genomic sequences we used in the analysis in a single convenient file.  
MopBMAGs.fasta contain every metagenomic sequence we gathered in a single convenient file.
MopBMAGs_50.fasta contain all the metagenomic MopB superfamily sequences chosen by CDHIT using the CDHIT-50 criteria.  These are the sequences used to generate Fig. 3.
allDMSORswoutMopaligned.fasta and MopBMAGs_50aligned.fasta contain all of the genomic sequences and metagenomic sequences used in our phylogenetic analyses after alignment by MAFFT.
All .dat files contain results of the sequence alignments from MopB superfamily, molybdenum hydroxylase, and aldehyde:ferredoxin oxidoreductase crystal structures.
Wellsetal_MopB_evolution_Sup_Table_1.xlsx contain all of the phyla counts from our dataset of 47,000 MopB superfamily sequences.
